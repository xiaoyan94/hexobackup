### 本地启动

```bash
hexo clean
hexo g -d
hexo s
```

## Hexo 升级

升级 hexo：`npm install hexo-cli -g`
安装插件：`npm install hexo-renderer-marked --save`
删除插件：`npm remove hexo-renderer-marked`
